## Challenge Prompt

I have built my own Git server with my own rules! 
You can clone the challenge repo using the command below. 
`git clone ssh://git@foggy-cliff.picoctf.net:56891/git/challenge.git` 
Check the `README` to get your flag!

## Problem Type
- Git

## Solve
We are told to pull down the Git repo so we will start with that:
<img width="888" height="316" alt="2026-03-22_17-31-46" src="https://github.com/user-attachments/assets/6d6bf9df-e4e2-4bc6-a651-bf0bc7195dff" />

Next we are told to check the `README` to get the flag. Let's move into the `challenge` folder and `cat` the file out:
```md
# MyGit

### If you want the flag, make sure to push the flag!

Only flag.txt pushed by ```root:root@picoctf``` will be updated with the flag.

GOOD LUCK!
```

So we need to push a flag.txt file as `root:root@picoctf`. Let's start by making a new `flag.txt` file.<br>
We will use `touch` to make a new file.<br>


Now we need to set our username and email address to make the push with:
```bash
# Set the Username
git config user.name "root"

# Set the email address
git config user.email "root@picoctf"

# Verify set correctly
git config --list
```
<img width="636" height="379" alt="2026-03-22_17-49-08" src="https://github.com/user-attachments/assets/48cc10c9-5ce1-498d-b0cd-dba5440db716" />



Next we need to add the file, and push the commit:
```bash
# Add all new files
git add .

# Commit the staged files to local repo
git commit -m "Add flag.txt"

# Push to remote repo
git push origin HEAD
```
We will re-enter the password and then we get the flag!
